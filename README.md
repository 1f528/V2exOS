## 关于

- 目标打造一个 macOS 平台原生极致体验的 V2ex 客户端
- 使用 SwiftUI 开发，尽量利用系统和语言新特性，最低适配 macOS 12+
- 开发过程中抽象出可复用的 V2ex API 网络层为 Swift Package - [V2exAPI](https://github.com/isaced/V2exAPI) 
- 目前还是早期预览版本，核心功能还处于开发中
